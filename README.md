# AI Chat App (Next.js)

This project is an AI-powered chat application built using Next.js, 
Framer Motion, and OpenAI API.

## Features
- Animated AI UI
- API-based AI responses
- Loading states & error handling
- Clean separation of frontend & backend

## Setup
1. Clone the repo
2. Create `.env.local`
3. Add your OpenAI API key:
   OPENAI_API_KEY=your_key_here
4. Run:
   npm install
   npm run dev

## Note
This project uses OpenAI API (paid).
If API quota is exceeded, responses may not work without billing.
